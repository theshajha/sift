# Sift

This folder is an agent definition, not an app. The judgment lives in plain markdown.

Start with `instructions.md`. The scoring rubric is in `skills/screen/skill.md`, and the
output shape is in `schema.md`. Point yourself at a folder under `roles/` (or the worked
example in `roles.example/`) and rank it.

There is nothing to install and no tools to wire up. Read the files with whatever you already
have.
