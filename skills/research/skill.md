# Research

Off by default. Run this only when the operator asks, and only for the people worth their
time.

Read only the URLs the candidate themselves provided (GitHub, portfolio, personal site).
Never search for them. Never guess private details. Write at most three short notes that
would help the operator decide: what they built, how deep it goes, one flag if there is one.
If a link is dead or empty, say so and move on.
