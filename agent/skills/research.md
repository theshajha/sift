description: Use for shortlisted applicants to read the public links they provided.
---

Only for worth_your_time or maybe applicants, and only when research is enabled.
Fetch ONLY the URLs in the applicant's `links` (GitHub, portfolio) with fetch_url.
Never search for them. Never guess private details. Write at most three short notes
that would help the operator decide (what they built, depth, one flag if any). If a
link is dead or empty, say so and move on.
